## Default Permission

Default permissions for skim-ai (MLX + Apple Foundation Models + iOS Keychain)

#### This default permission set includes the following:

- `allow-mlx-is-available`
- `allow-mlx-is-model-downloaded`
- `allow-mlx-download-model`
- `allow-mlx-delete-model`
- `allow-mlx-complete`
- `allow-fm-is-available`
- `allow-fm-availability`
- `allow-fm-complete`
- `allow-ios-keychain-store`
- `allow-ios-keychain-load`
- `allow-ios-keychain-clear`
